## [0.1.0] — 2026-06-22

### Initial Release

This is the first major restructuring and documentation of the Dopoamind project with a new organized folder structure.

### Added

#### Documentation Structure
- Created `/docs/` folder for organized documentation
- Created `/docs/dopamine-neurobiology/` section with comprehensive dopamine science
  - `README.md` — Section overview
  - `synthesis-and-metabolism.md` — Detailed guide to dopamine synthesis (tyrosine pathway) and metabolism (MAO, COMT)
  - `receptors.md` — Comprehensive dopamine receptor reference (D1–D5; structure, function, signaling)
- Created `/docs/withdrawal-protocols/` section on withdrawal and recovery
  - `README.md` — Section overview with safety disclaimers
  - `withdrawal-timeline.md` — Detailed timeline of stimulant withdrawal from hours to months; symptoms, neurobiological mechanisms, and recovery strategies
- Created `/docs/chemicals/` section for substance information
  - `README.md` — Overview and organization; safety disclaimers

#### Major Documentation Files
- **INDEX.md** — Comprehensive searchable index with topic and question-based navigation
- **CONTRIBUTING.md** — Detailed guidelines for contributors, including:
  - Style guide and markdown formatting standards
  - Citation and reference guidelines
  - Content organization and review process
  - Core principles (evidence-based, educational, safety-first)
- **CHANGELOG.md** — This file; tracks all changes and additions

#### Existing Content
- **README.md** — Maintained original project overview and introduction (updated)
- **References.md** — 12 peer-reviewed citations on dopamine and addiction (maintained)
- **LICENSE** — MIT License (maintained)

### Improved
- Better organization of documentation for findability and usability
- Enhanced safety disclaimers and crisis resource information
- Clearer navigation with INDEX.md and internal linking
- Standardized structure across all documentation sections
- Added detailed table of contents and section overviews

### Documentation Coverage

Total files: 12  
Total peer-reviewed references: 12+  
Major topics covered:
- Dopamine synthesis and metabolism (tyrosine → dopamine)
- Dopamine receptors (D1, D2, D3, D4, D5) and signaling cascades
- Neural circuits (mesolimbic, mesocortical, nigrostriatal, tuberoinfundibular)
- Stimulant withdrawal timeline and phases (acute, subacute, protracted)
- Withdrawal symptoms and neurobiological mechanisms
- Recovery timeline and processes (neuroplasticity, receptor recovery)
- Factors affecting withdrawal severity and recovery duration
- Safety information and crisis resources

### Standards Established
- Markdown formatting and style conventions
- Citation and reference standards
- Content scope and out-of-scope boundaries
- Safety disclaimer requirements
- Contribution review process

## How to Report Issues

Found an error, missing reference, or have a suggestion? Please:
1. Check [Issues](https://github.com/gasperrus-a11y/Dopoamind-ultimate-dopamine-enhancement/issues) to see if already reported
2. Open a new issue with:
   - Clear description of the problem or suggestion
   - Relevant file(s) and line numbers (if applicable)
   - Any supporting information or citations
3. Maintainers will respond to discuss and prioritize
